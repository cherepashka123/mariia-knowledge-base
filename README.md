# Mariia Knowledge Base API

This repository contains the knowledge base for Mariia Cherep, deployed as a Vercel API endpoint.

## API Endpoint

Once deployed, the JSON data will be available at:
- `https://your-project.vercel.app/api`

## Local Development

```bash
npm install
vercel dev
```

## Deployment

Deploy to Vercel:

```bash
vercel
```

Or connect your GitHub repository to Vercel for automatic deployments.
